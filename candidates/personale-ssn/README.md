# personale-ssn

Personale del Servizio Sanitario Nazionale per regione, ruolo e categoria (2010-2021).

**Fonte**: Ministero della Salute — Dati del Personale SSN ([dati.salute.gov.it](https://www.dati.salute.gov.it/))
**Licenza**: dati pubblici

## Domanda guida

Quanto personale lavora nel SSN? Come si distribuisce tra regioni, ruoli e generi? Come evolve nel tempo?

## Dataset

- **Copertura**: 2010–2021 (12 anni)
- **Granularità**: regionale per ruolo categoria
- **Righe**: ~18.500 (clean), ~1.250 (mart)
- **Colonne clean**: 15 (prospetto, anno, regione, azienda, ruolo, dotazioni, tempo pieno/part-time per genere)
- **Personale SSN**: ~650.000 unità (2021)

## Stato

`candidate` — run verificato su anno 2021 (serie 2010-2021 completa).
