# elezioni-europee

Risultati delle elezioni del Parlamento Europeo in Italia per comune e lista (1979-2024).

**Fonte**: Eligendo — Archivio storico elettorale del DAIT (Ministero dell'Interno)
**Licenza**: dati pubblici

## Domanda guida

Come è cambiato il voto degli italiani alle elezioni europee dal 1979 a oggi? Quali liste hanno guadagnato o perso consensi nel tempo?

## Dataset

- **Copertura**: 1979–2024 (10 tornate)
- **Granularità**: comunale per lista
- **Righe**: ~1,2 milioni
- **Colonne**: 9 (data_elezione, circoscrizione, regione, provincia, comune, lista, voti_lista, elettori, votanti)
- **Affluenza storica**: 85% (1979) → 50% (2024)

## Stato

`candidate` — run verificato su tutte le 10 tornate.
