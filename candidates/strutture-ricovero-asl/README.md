# strutture-ricovero-asl

Dataset del Ministero della Salute — Strutture di ricovero pubbliche presenti nel territorio della ASL.

**Fonte**: https://www.dati.salute.gov.it/it/dataset/strutture-di-ricovero-pubbliche-e-equiparate-presenti-nel-territorio-della-asl
**URL diretto CSV**: `Strutture_di_ricovero_pubbliche_presenti_nel_territorio_della_ASL.csv`
**Anno**: 2022
**Licenza**: CC BY 4.0

**Domanda**: Dotazione ospedaliera (personale, posti letto, ricoveri) per ASL e regione.

**Output**: `mart_regioni.parquet` — personale, medici, infermieri, posti letto per regione.
