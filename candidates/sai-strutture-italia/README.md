# sai-strutture-italia

Strutture del Sistema Accoglienza e Integrazione (SAI) per migranti in Italia — appartamenti, comunità alloggio e centri collettivi.

**Fonte**: Centri d'Italia (ActionAid Italia + Fondazione Openpolis)
**Licenza**: CC-BY 4.0

## Domanda

Quali e quante strutture SAI ci sono in Italia, dove si trovano, di che tipo, con che capienza? Come si distribuiscono per regione e anno?

## Dataset

Singola struttura SAI per rilevazione annuale. File unico multi-anno.

- **Copertura**: 2018–2024
- **Granularità**: singola struttura per rilevazione (~46.000 righe)
- **Tipologie**: APPARTAMENTO (83%), COMUNITÀ ALLOGGIO, CENTRO COLLETTIVO, ecc.

## Output minimo atteso

- Mart per regione × anno × tipologia (capienza, presenze)
- Notebook di confronto con CAS (centri-accoglienza-italia)
