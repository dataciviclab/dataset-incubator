# sai-progetti-italia

Progetti del Sistema Accoglienza e Integrazione (SAI) per migranti in Italia. Ogni riga è un progetto finanziato a un ente locale, con tipologia (ordinario, minori, dimissioni), capienza e presenze.

**Fonte**: Centri d'Italia (ActionAid Italia + Fondazione Openpolis)
**Licenza**: CC-BY 4.0

## Domanda

Quanti progetti SAI ci sono in Italia, chi li gestisce, quanti posti hanno, per regione e tipologia?

## Dataset

- **Copertura**: 2018–2024
- **Granularità**: singolo progetto per rilevazione annuale (~6.100 righe)
- **Tipologie**: ORD (ordinario) ~73%, MSNA (minori) ~22%, DM-DS ~5%
- **Metriche**: capienza, presenze, ente locale gestore
- **Capienza media**: ORD ~50 posti, MSNA ~28, DM-DS ~17

## Output minimo atteso

- Mart per regione × anno × tipologia (numero progetti, capienza, presenze)
- Confronto con strutture SAI per capienza media progetto
