# strutture-asl

Dataset del Ministero della Salute — Strutture e attività delle ASL.

**Fonte**: https://www.dati.salute.gov.it/it/dataset/strutture-e-attivita-asl/
**URL diretto CSV**: `Strutture_e_attività_ASL.csv`
**Anno**: 2022
**Licenza**: CC BY 4.0

**Domanda**: Dotazione territoriale di medici di base e pediatri per ASL.

**Output**: `mart_regioni.parquet` — medici, pediatri, popolazione residente per regione.
