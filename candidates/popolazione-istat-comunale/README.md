# Popolazione ISTAT Comunale

Questo dataset contiene dati sulla popolazione comunale forniti da ISTAT per gli anni 2019‑2025.

## Descrizione

Il dataset fornisce informazioni demografiche a livello di comune, includendo:
- Popolazione totale
- Suddivisione per genere
- Altri indicatori demografici

## Utilizzo

Vedi `notes.md` per dettagli su come eseguire le query e interpretare i risultati.
